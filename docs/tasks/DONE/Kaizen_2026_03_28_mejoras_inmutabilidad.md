# Tarea Kaizen: Mejoras de Inmutabilidad

Durante la revisión, se ha detectado que hay varios Handlers de MediatR que continúan utilizando 'class' en lugar de 'record', y que se pueden mejorar otros aspectos de inmutabilidad en la capa de Aplicación.

El objetivo de esta tarea es mejorar la robustez de la base de código aplicando principios de inmutabilidad, en consonancia con las mejores prácticas arquitectónicas de SddIA.
