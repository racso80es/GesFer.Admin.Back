# Actualizaciones Automáticas con Dependabot

## Descripción
El ciclo de vida del software implica que dependencias utilizadas hoy pueden revelar fallos de seguridad críticos (CVEs) mañana.

## Automatización
Herramientas como Dependabot monitorean constantemente las librerías del proyecto. Cuando se detectan vulnerabilidades conocidas en la cadena de suministro, estas herramientas automatizan el proceso de actualización disparando *Pull Requests* para resolver los problemas y parchear el código lo antes posible, reduciendo drásticamente la ventana de ataque del ecosistema [30, 31].
