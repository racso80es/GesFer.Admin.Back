# Skill: Frontend Unit Testing

**skill_id:** `frontend-test`

## Objetivo

Comandos estandarizados para ejecutar tests unitarios en proyectos Frontend (Desktop, Product, Admin).

## Scripts

- test-desktop-unit: Vitest para Kalma2 Desktop (npm run test:run)
- test-desktop-coverage: Vitest coverage (npm run test:coverage)
- test-desktop-e2e: Playwright E2E (npm run e2e; requiere Display/Xvfb)
